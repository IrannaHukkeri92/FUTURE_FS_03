# FitZone Gym - Modern Fitness Website 🏋️‍♂️

## 📋 Project Overview

A **modern, professional, and fully responsive website** for a local gym business designed to attract customers, improve online presence, and drive business growth.

---

## 🛠 Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4
- **Icons**: Lucide React
- **Fonts**: Google Fonts (Inter)
- **Images**: Unsplash API

---

## 🎨 Design Features

### ✨ Visual Elements
- **Clean, Modern Layout**: Professional gradient color scheme (orange → yellow accent colors)
- **Responsive Design**: Mobile-first approach, works on all devices
- **Smooth Animations**: Hover effects, transitions, scroll animations
- **Glass Morphism**: Backdrop blur effects
- **Custom Scrollbar**: Orange-themed scrollbar
- **Shadow Effects**: Multi-layer shadows for depth

### 🎯 Color Palette
- **Primary**: Orange gradients (#f97316 → #fbbf24)
- **Background**: Gray scale (#f9fafb, #ffffff, #111827)
- **Text**: Dark gray (#1f2937) for light backgrounds, white for dark
- **Accents**: Blue, green, purple for info cards
